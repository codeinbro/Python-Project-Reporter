# Python Project Reporter

Python Project Reporter is an automated tool to generate a complete report of your Python project.  
It produces a report including folder structure, file list, code content, and report date. Perfect for documentation, presentation, or AI-assisted project analysis.

---

## Key Features

- Generates **Project Tree** for `.py` files (can be extended)
- Includes **all Python code files** in the report
- Automatically adds **report date**
- Skips cache or generator files
- Output is saved as **`project_report_output.txt`** in the `docs/` folder
